# 📊 Purchase order Dashboard (Looker Studio)

## 📌 Overview

This project showcases an interactive **Sales Order & Order Punch Dashboard** built using Google Looker Studio. It provides real-time insights into sales performance, order trends, and business KPIs.

---

## 🚀 Key Features

* 📈 **Total Orders & Sales Quantity Tracking**
* 🌍 **Geographical Sales Distribution (Map View)**
* 📊 **Monthly Sales Quantity Analysis (Year-wise)**
* 💰 **Conversions & Revenue Breakdown**
* 👨‍💼 **Salesperson Performance Analysis**
* 🛍 **Top Product Sales Overview**
* 🏢 **Highest Purchase Party Insights**
* 🔍 **Dynamic Filters (Order No, Party Name, Date, Article, etc.)**

---

## 🛠 Tools & Technologies

* Google Looker Studio
* Google Sheets (Data Source)
* Data Visualization Techniques

1. Data is maintained in Google Sheets

3. Dashboard auto-updates with filters and controls
4. Provides actionable insights for business decisions
